# Minishell
The project goal is to recode a mini UNIX shell in C language.

## Allowed functions:

- write
- opendir, readdir, closedir
- stat, lstat, fstat
- malloc, free
- exit
- getcwd
- chdir
- fork
- open, close, read
- execve
- access
- wait, waitpid, wait3, wait4
- signal
- kill


## Bonus

- Signal management : (ctrl-C)
- Rights management
- Autocompletion
- Commands separation with ';'
